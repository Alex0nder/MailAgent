MCP signup QA

Connect the MailAgent MCP server and call mailagent_verify_signup with a service preset (GitHub, GitLab, Google, …). The tool returns agent.primaryAction — OTP or magic link — without parsing HTML manually.

Recommended flow

  1. mailagent_create_inbox with service=github
  2. Submit address on the signup form (browser automation)
  3. mailagent_verify_signup with inboxId
  4. On timeout: response includes debugUiUrl or mailagent_diagnose_inbox

Install

npx -y -p @mailagent/mcp@latest mailagent-mcp
# or remote: POST https://api.webmailagent.com/mcp + Bearer

npx skills add Alex0nder/MailAgent --skill mailagent
