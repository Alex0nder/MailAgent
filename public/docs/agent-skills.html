Agent Skills

MailAgent ships a canonical skill for Cursor, Codex, Claude Code, and npx skills add — signup OTP and magic links without parsing HTML.

Install

npx skills add Alex0nder/MailAgent --skill mailagent

Source: skills/mailagent/SKILL.md

After editing the skill

npm run sync:skills
npm run verify:skills

Works with integration skills

MailAgent handles verification email during signup. After login, use app skills (e.g. Membrane application-skills for GitHub, Slack, Jira).

  1. mailagent_create_inbox (service: github)
  2. Submit address on signup form
  3. mailagent_verify_signup → OTP or magic link
  4. Membrane github skill → issues, PRs, etc.
