# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| latest on `main` | yes |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security-sensitive reports.

1. Use [GitHub private vulnerability reporting](https://github.com/Alex0nder/MailAgent/security/advisories/new) on this repository, or
2. Email the maintainer via the contact on [webmailagent.com](https://webmailagent.com).

Include steps to reproduce, impact, and affected components (API, MCP, dashboard, Codex plugin).

We aim to acknowledge reports within 3 business days.
